üìö About LinOSS

LinOSS (Linear Oscillating State Space) is a novel neural architecture designed for modeling complex dynamical systems. This web demo showcases real-time simulation of coupled neural oscillators with interactive parameter control and visualization.

The D-LinOSS variant uses diagonal parameterization for efficient computation while maintaining the expressive power needed for neural signal modeling and brain dynamics simulation.
